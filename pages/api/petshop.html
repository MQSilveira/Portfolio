Petshop API in Node JS

The PetShop API is a Node.js application designed to streamline the management of clients, dogs, and appointments within a pet shop environment. This API provides a comprehensive set of routes for interacting with client information, dog details, and appointment records. With user-friendly endpoints, it offers the flexibility needed to efficiently handle the day-to-day operations of a pet shop.
